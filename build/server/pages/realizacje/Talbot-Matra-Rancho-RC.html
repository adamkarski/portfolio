Talbot Matra Rancho RC



Originally TALBOT MATRA RANCHO has 27.125 MHz analog controller.

Custom firmware for RC controller in arduino, operating on wireless protocol engineered by Espressif Systems named ESP-NOW. Controller is fully proportional with four analog potentiometers and has tested range 430m. It has integrated 132x32 oled display that shows the current direction of turn in percent and acceleration. In addition, the display shows parameters such as the activation of automatic turn signals, high beams, dipped beams, etc.

An additional "Menu" button allows you to enter the menu system and set these parameters for the model. And changing these parameters for the model by using direction and acceleration switches.

Model is operating with NodeMcu V3 board (ESP8266 chip) and L293D Motor Shield.

After connecting the transmitter to the USB port, access to the menu via the Serial console is available.

Future version will have option to control multiple models.

More info and code : github.com/toshinori8
